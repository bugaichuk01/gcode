# Знаки соответствия (SVG для canvas и превью)

**Плейсхолдеры:** файлы в этой папке — временные заглушки с аббревиатурой знака в рамке.
Их нужно заменить на официальную графику знаков соответствия **без изменения имён файлов**:

- `rst_decl.svg` — РСТ декларирования
- `ctr.svg` — СТР
- `rst.svg` — РСТ
- `eac.svg` — ЕАС
- `ce.svg` — CE

Реестр: `../signRegistry.ts`.
